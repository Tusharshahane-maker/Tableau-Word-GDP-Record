Overview
This repository hosts the data and code used to generate the GDP Map Visualization for the year 2014. This project utilizes a choropleth map to visually represent economic data, specifically Gross Domestic Product (GDP), across various countries globally. The visualization aims to provide a quick, comparative insight into the global economic landscape of that year.

Key Features
Choropleth Map: Displays country-level data using color-coding.

Global Coverage: Includes data for numerous countries and regions worldwide.

Time-Specific Data: Focuses on the economic record for the year 2014.
